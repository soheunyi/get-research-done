# Ablation Output Contract

Output sections:
1. Baseline and evaluation target
2. Ranked ablation table
3. Sequenced execution batches
4. Expected signal per batch
5. Stop criteria and fallback plan

## Ranked Table Columns
- id
- change vs baseline
- expected signal
- confidence
- cost (low/med/high)
- priority

## Artifact
Write `.grd/research/ABLATION_PLAN.md` when requested.
